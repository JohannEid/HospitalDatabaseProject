1. Donne le nom et prenom des malades ayant pour mutuelle la LMDE:
SELECT nom, prenom FROM malade WHERE mutuelle='LMDE'

2. Modifier des données
UPDATE malade
SET mutuelle = 'LMDE',
WHERE numero = '1',
  nom = 'Querrey',
  prenom = 'Sam'

3. Supprimer
DELETE FROM `utilisateur`
WHERE `date_inscription` < '2012-04-10'
